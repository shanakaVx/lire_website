Domain What is Lire

Domain of Lire

Tone based voice synthersizer framework for voice based application development

Lire = "Read" in French

Features of Lire

Lire is an effective voice synthesizer framework

Lire is based on Sinhala language, which contains the phonetics for Sinhala and Extended Phonetic Alphabet.

Editable speech timeline

Editable text to speech rules and prosodies.

User’s own voice can be fed into the system to generate voice from the text.

API where all the Lire services are exposed
